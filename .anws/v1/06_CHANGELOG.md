# Changelog: v1

## Summary

Initial architecture document set for `玩点啥.ai`, a lightweight random website toy recommender for offline AI experience events.

## Changes

- Created first versioned architecture directory at `.anws/v1`.
- Defined the product scope as a no-login, no-backend, local-data web demo.
- Preserved implementation as a later `/blueprint` and `/forge` concern.
- Excluded AI smart recommendation from v1 and recorded it as a future-version decision point.
- Added blueprint execution plan `05A_TASKS.md` and verification plan `05B_VERIFICATION_PLAN.md`.
- Closed challenge findings CH-01 through CH-08 by adding minimal system designs and tightening 05A/05B verification contracts.

## Compatibility

- No previous architecture version exists.
